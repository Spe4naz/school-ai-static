# School AI - Интеллектуальная образовательная платформа

Добро пожаловать в вики проекта **School AI** -- полноценной онлайн-школы с электронным дневником, расписанием, чатом и системой уведомлений.

---

## Что умеет платформа

| Возможность | Описание |
|-------------|----------|
| **Электронный дневник** | Выставление и просмотр оценок по предметам с фильтрацией по неделям |
| **Расписание** | Расписание уроков для классов с CRUD-операциями |
| **Классный чат** | Чат с клиентским AES-256-GCM шифрованием сообщений |
| **Домашние задания** | Назначение и просмотр домашних заданий |
| **Объявления** | Школьные объявления от учителей и администрации |
| **Уведомления** | Реалтайм-уведомления через SSE (Server-Sent Events) |
| **Отчёты** | Экспорт оценок в PDF и Excel |
| **Админ-панель** | Управление пользователями, классами, кодами, бэкапами и настройками |
| **Сброс пароля** | Восстановление пароля через email |
| **Автобэкапы** | Ежечасные автоматические резервные копии БД |

---

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Администратор** | Полный доступ: управление пользователями, классами, кодами, бэкапами, статистика |
| **Старший учитель** | Доступ к управлению пользователями (ограниченно) |
| **Учитель** | Выставление оценок, создание расписания, домашних заданий, объявления |
| **Ученик** | Просмотр оценок, расписания, чата, домашних заданий |
| **Родитель** | Просмотр данных привязанного ученика |

---

## Стек технологий

```
Backend:   Node.js + Express + TypeScript
Database:  PostgreSQL 16
Frontend:  Vanilla JS (ES Modules) + Chart.js
Proxy:     Caddy (авто-SSL)
Тесты:     Jest + Supertest + Testcontainers
Деплой:    Docker + Docker Compose
Безопасность: helmet + CSP nonces, JWT в httpOnly cookies, rate-limit (IP + per-user)
```
